How to Generate JIT Symbol Files

Any tool that can generate the format described here will work.

With Node.js >=v0.11.3 do the following:

node --perf-basic-prof your-app.js
and ensure to exit your process cleanly or use flush-all to to ensure that your map file is written completely.

The map file is created at /tmp/perf-<pid>.map.
